Intelligent IT Monitoring. Coming soon to Open-Source

Recon helps DevOps professionals monitor and manage their infrastructure smartly by using intelligent defaults, automatic analytics & predictive reporting.

Completely open-source, Recon's goal is to offer comprehensive SaaS quality monitoring with added emphasis on Intuitiveness, Automation and Usability.

Key Features:
  • Default configuration for Single command install
  • Process discovery services and recommendations
  • Analytics engine that creates automatic dashboards
  • Automatic plugins installations
  • Machine learning and trend predictions
  • Real time and predictive alerting
